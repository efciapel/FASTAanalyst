Witaj w dokumentacji pakietu FASTAanalyst!¶

fa.alignment(dna_1, dna_2, gap)[źródło]¶

Zwraca nić score przyrównania dwóch sekwencji. Wykorzystuje algorytm Needlemana-Wunscha. Args:

dna_1 (str): pierwsza sekwencja nukleotydowa. dna_2 (str): druga sekwencja nukleotydowa. gap (int): kara za przerwę
fa.complement(seq)[źródło]¶

Zwraca nić komplementarną sekwencji. Args:

seq (str): sekwencja nukleotydowa.
fa.length(seq)[źródło]¶

Zwraca długość podanej sekwencji. Args:

seq (str): sekwencja nukleotydowa.
fa.match(base_1, base_2)[źródło]¶

Sprawdza czy dane nukleotydy są takie same i zwraca odpowiednią wartość Args:

base_1 (str): pierwszy nukleotyd. base_2 (str): drugi nukleotyd.
fa.melting_point(seq)[źródło]¶

Zwraca temperaturę topnienia sekwencji. Args:

seq (str): sekwencja nukleotydowa.
fa.molecular_mass(dna)[źródło]¶

Zwraca nić masę cząsteczkową łańcucha. Args:

dna (str): sekwencja nukleotydowa.
fa.nucleotide_occurrence(dna, *args)[źródło]¶

Podaje procentowe występowanie danego nukleotydu w sekwencji lub grupy nukleotydów. Args:

dna (str): sekwencja nukleotydowa.
fa.read_dna(dna)[źródło]¶

Odczytuje sekwencje DNA. Args:

dna (str): sekwencja nukleotydowa.
fa.read_fasta(fasta_file)[źródło]¶

Odczytuje plik FASTA. Args:

fasta_file (str): ścieżka pliku FASTA.
fa.reverse_complement(seq)[źródło]¶

Zwraca odwróconą nić komplementarną sekwencji. Args:

seq (str): sekwencja nukleotydowa.
